#Application de gestion des annonces de vente de voiture en ligne



Pour un utilisateur tiers, il lui suffit juste de cliquer sur le lien de l'application en ligne : 
   1. Pour consulter les annonces en ligne, il suffit de cliquer sur le menu Produit en haut de la page d'acceuil
   2. Pour consuter les détails d'une annonce, if faudra cliquer sur détail toujours en haut de la paqe d'acceuil 



Pour une personne désirant publié une annonce en ligne, il lui faudra:
   1. Accéder a l'application via le lien de l'application
   2. Créer un compte en cliquant sur inscription fournisseur
   3. Se connecter en cliquant sur le bouton connecter
